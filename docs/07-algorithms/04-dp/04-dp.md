# 动态规划

> 考察频率：★★★★★  题型识别：最优子结构 + 重叠子问题

## DP 解题四步法

1. **定义状态**：`dp[i]` 是什么
2. **状态转移**：`dp[i] = f(dp[j])` 如何推导
3. **初始化**：base case
4. **遍历顺序**：保证计算 dp[i] 时 dp[j] 已就绪

### 什么题该用 DP
- 计数型：有多少种方式
- 最值型：最大值/最小值
- 存在型：能不能、是否可行

## 题型一：0/1 背包（LeetCode 416）

### 题目
判断数组能否分成两个和相等的子集。

### 思路
等价于：能否从数组中选若乾元素使其和为 sum/2 → 0/1 背包问题。

- `dp[j]` = 是否能恰好装满容量 j
- 状态转移：`dp[j] = dp[j] || dp[j-nums[i]]`

```go
func canPartition(nums []int) bool {
    sum := 0
    for _, v := range nums {
        sum += v
    }
    if sum%2 == 1 {
        return false
    }
    target := sum / 2

    // dp[j] = 是否能凑出和 j
    dp := make([]bool, target+1)
    dp[0] = true

    for _, v := range nums {
        // 必须从后往前！0/1 背包每个物品只能用一次
        for j := target; j >= v; j-- {
            dp[j] = dp[j] || dp[j-v]
        }
    }
    return dp[target]
}
```

### 一维 DP 的本质
二维写法：`dp[i][j] = dp[i-1][j] || dp[i-1][j-v]` → 空间优化后变成一维，因为只依赖上一行，用从后往前遍历避免覆盖。

## 题型二：完全背包（LeetCode 322 - 零钱兑换）

### 题目
给定硬币面额，每种硬币无限使用，求凑成 amount 的最少硬币数。

### 代码

```go
func coinChange(coins []int, amount int) int {
    // dp[j] = 凑成金额 j 的最少硬币数
    dp := make([]int, amount+1)
    for i := range dp {
        dp[i] = amount + 1 // 初始化为不可能的大数
    }
    dp[0] = 0

    for _, coin := range coins {
        // 完全背包：正序遍历（每个物品可重复选）
        for j := coin; j <= amount; j++ {
            dp[j] = min(dp[j], dp[j-coin]+1)
        }
    }

    if dp[amount] == amount+1 {
        return -1
    }
    return dp[amount]
}
```

**0/1 背包 vs 完全背包**：
| 类型 | 遍历顺序 | 说明 |
|------|---------|------|
| 0/1 背包 | `for j := target; j >= v; j--` | 每个物品用一次，倒序保证不重复 |
| 完全背包 | `for j := v; j <= target; j++` | 每个物品可用无限次，正序 |

## 题型三：最长公共子序列（LeetCode 1143）

### 题目
给定两个字符串，求最长公共子序列的长度。

### 代码

```go
func longestCommonSubsequence(text1 string, text2 string) int {
    m, n := len(text1), len(text2)
    // dp[i][j] = text1[0..i-1] 和 text2[0..j-1] 的 LCS 长度
    dp := make([][]int, m+1)
    for i := range dp {
        dp[i] = make([]int, n+1)
    }

    for i := 1; i <= m; i++ {
        for j := 1; j <= n; j++ {
            if text1[i-1] == text2[j-1] {
                dp[i][j] = dp[i-1][j-1] + 1
            } else {
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])
            }
        }
    }
    return dp[m][n]
}
```

### 追问：如何输出 LCS 字符串？
→ 从 dp[m][n] 倒推，相等则加入字符串，否则往大的方向走。

## 题型四：最长递增子序列（LeetCode 300）

### 题目
给定数组，找最长递增子序列的长度。

### 思路一：O(n²) DP

```go
func lengthOfLIS(nums []int) int {
    if len(nums) == 0 {
        return 0
    }
    dp := make([]int, len(nums))
    for i := range dp {
        dp[i] = 1
    }

    for i := 0; i < len(nums); i++ {
        for j := 0; j < i; j++ {
            if nums[j] < nums[i] {
                dp[i] = max(dp[i], dp[j]+1)
            }
        }
    }

    result := 0
    for _, v := range dp {
        result = max(result, v)
    }
    return result
}
```

### 思路二：O(n log n) 二分优化（Greedy + 二分）

维护一个 `tails` 数组：`tails[i]` = 长度为 i+1 的递增子序列的最小尾部值。

遍历 nums，用二分找到第一个 >= num 的位置更新，插入位置即为以 num 结尾的最长递增子序列长度。

```go
func lengthOfLIS(nums []int) int {
    tails := make([]int, 0)
    for _, num := range nums {
        // 找 tails 中第一个 >= num 的位置
        i := sort.Search(len(tails), func(i int) bool {
            return tails[i] >= num
        })
        if i == len(tails) {
            tails = append(tails, num)
        } else {
            tails[i] = num
        }
    }
    return len(tails)
}
```

## 题型五：编辑距离（LeetCode 72）

### 题目
把 word1 变成 word2 最少需要几次操作（插入、删除、替换）。

### 代码

```go
func minDistance(word1 string, word2 string) int {
    m, n := len(word1), len(word2)
    // dp[i][j] = word1[0..i-1] 变成 word2[0..j-1] 的最少操作数
    dp := make([][]int, m+1)
    for i := range dp {
        dp[i] = make([]int, n+1)
        dp[i][0] = i // 删除 i 个字符
    }
    for j := 0; j <= n; j++ {
        dp[0][j] = j // 插入 j 个字符
    }

    for i := 1; i <= m; i++ {
        for j := 1; j <= n; j++ {
            if word1[i-1] == word2[j-1] {
                dp[i][j] = dp[i-1][j-1]
            } else {
                dp[i][j] = min(
                    dp[i-1][j]+1,   // 删除 word1[i-1]
                    dp[i][j-1]+1,   // 插入 word2[j-1]
                    dp[i-1][j-1]+1, // 替换
                )
            }
        }
    }
    return dp[m][n]
}
```

## 题型六：状态压缩 DP（LeetCode 698 - 划分为 k 个相等子集）

### 题目
数组能否分成 k 个和相等的子集。

### 思路
用 bitmask 表示已使用的数字。`dp[mask]` = 是否能恰好划分，状态转移枚举最后一个加入的子集。

```go
func canPartitionKSubsets(nums []int, k int) bool {
    sum := 0
    for _, v := range nums {
        sum += v
    }
    if sum%k != 0 {
        return false
    }
    target := sum / k
    n := len(nums)
    sort.Ints(nums)
    // 提前剪枝：最大的数超过 target 直接 false
    if nums[n-1] > target {
        return false
    }

    // dp[mask] = mask 表示的集合能否成功划分
    totalMask := 1 << n
    dp := make([]int8, totalMask)
    dp[0] = 1 // dp[mask] = 1 表示可划分，-1 表示不可划分，0 表示未计算

    var dfs func(mask int) bool
    dfs = func(mask int) bool {
        if dp[mask] != 0 {
            return dp[mask] == 1
        }
        dp[mask] = -1 // 标记为正在计算（防止环）

        sumMask := 0
        for i := 0; i < n; i++ {
            if mask&(1<<i) == 0 {
                sumMask += nums[i]
            }
        }
        curSum := sumMask % target

        for i := 0; i < n; i++ {
            if mask&(1<<i) == 0 && curSum+nums[i] <= target {
                if dfs(mask | (1 << i)) {
                    dp[mask] = 1
                    return true
                }
            }
        }
        dp[mask] = -1
        return false
    }

    return dfs(0)
}
```

## 题型七：打家劫舍系列（LeetCode 198/213/337）

### 打家劫舍 I（线性）

```go
func rob(nums []int) int {
    if len(nums) == 0 {
        return 0
    }
    if len(nums) == 1 {
        return nums[0]
    }
    dp := make([]int, len(nums))
    dp[0] = nums[0]
    dp[1] = max(nums[0], nums[1])
    for i := 2; i < len(nums); i++ {
        dp[i] = max(dp[i-1], dp[i-2]+nums[i])
    }
    return dp[len(nums)-1]
}
```

### 打家劫舍 II（环形，首尾不能同时偷）

```go
func rob2(nums []int) int {
    if len(nums) == 0 {
        return 0
    }
    if len(nums) == 1 {
        return nums[0]
    }
    // 两种情况：抢 [0..n-2] 或抢 [1..n-1]
    return max(robRange(nums, 0, len(nums)-2), robRange(nums, 1, len(nums)-1))
}

func robRange(nums []int, start, end int) int {
    if start == end {
        return nums[start]
    }
    prev, curr := nums[start], max(nums[start], nums[start+1])
    for i := start + 2; i <= end; i++ {
        prev, curr = curr, max(curr, prev+nums[i])
    }
    return curr
}
```

### 打家劫舍 III（树形）

```go
func rob(root *TreeNode) int {
    result := robTree(root)
    return max(result[0], result[1])
}

// 返回 [notRob, rob] 当前子树不偷/偷能获得的最大值
func robTree(node *TreeNode) [2]int {
    if node == nil {
        return [2]int{0, 0}
    }
    left := robTree(node.Left)
    right := robTree(node.Right)

    notRob := max(left[0], left[1]) + max(right[0], right[1])
    rob := left[0] + right[0] + node.Val
    return [2]int{notRob, rob}
}
```

## 常见 DP 模型总结

| 模型 | 状态定义 | 核心转移 |
|------|---------|---------|
| 0/1 背包 | `dp[j]` 是否能凑成 j | `dp[j] \|\| dp[j-v]` |
| 完全背包 | `dp[j]` 最少/最多硬币数 | `dp[j] = min/max(dp[j], dp[j-v]+1)` |
| 子序列 LCS | `dp[i][j]` 前缀匹配长度 | 相等=左上+1，不等=max(上,左) |
| 线性 DP | `dp[i]` 以 i 结尾的最优 | 枚举上一个状态 |
| 区间 DP | `dp[i][j]` 区间 [i,j] 最优 | 枚举分割点 k |
| 状态压缩 | `dp[mask]` 已选集合状态 | 枚举子集或添加元素 |
| 树形 DP | `dp[node][0/1]` 选或不选 | 递归合并子节点结果 |

## 面试高频追问

1. **如何优化 DP 空间**？一维能优化吗？→ 看状态转移依赖哪几行
2. **初始化太复杂怎么办**？→ 用 `dp = make([]type, size)` + 显式初始化 base case
3. **如何输出具体方案**？→ 记录每个状态的决策来源，从终点回溯
4. **股票买卖问题**？→ 5 种变种，统一用状态机 DP：`dp[i][k][0/1]` = 第 i 天，交易 k 次，持有/不持有
