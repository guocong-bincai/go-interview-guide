# 薪资谈判：锚点设置、竞争 Offer 利用、谈 Offer 技巧

> 考察频率：★★★★☆  难度：★★★☆☆
> 关键词：总包、Base、期权、RSU、谈判时机、锚点

---

## 核心答案（30 秒版）

**谈 Offer 的核心原则**：
1. 永远要谈，不要直接 Accept 第一个 Offer
2. 用竞争 Offer 做筹码（最好有 2 家以上）
3. 先谈 Base 再谈其他（Base 最难调，Sign-on 最容易）
4. 谈的是**总包**，不是月薪

---

## 1. 薪酬构成拆解

| 组成部分 | 说明 | 可谈空间 |
|----------|------|----------|
| **Base（基本工资）** | 月薪 × 12 或 16 | 难（公司有宽带限制）|
| **Bonus（年终奖）** | 通常 0-6 个月 | 中（取决于公司）|
| **Sign-on（签约奖金）** | 一次性入职奖励 | 大（公司愿意给）|
| **RSU/期权** | 股票归属 | 中（看公司政策）|
| **福利** | 社保、公积金、补贴 | 小 |

**总包计算**：
```
总包（年）= Base × 月数 + Bonus + Sign-on + RSU归属价值
```

> 例子：Base 40K × 16 + Bonus 10万 + Sign-on 5万 = 79万/年

---

## 2. 谈判时机

### ✅ 最佳时机

- 收到 Offer 后（但还没签字）
- 有竞争 Offer 时（最强筹码）
- 公司主动联系你时（他们有需求）

### ❌ 错误时机

- 面试还没结束就谈薪资
- 面试表现很差时强谈
- 已经签字后反悔（很伤信誉）

---

## 3. 谈判策略

### 3.1 第一步：了解市场

在谈之前，先搞清楚市场价：

```
如何估算：
1. 看准网/脉脉/Lever.io 查目标公司薪资范围
2. 问同级别朋友（非敏感，但可以模糊问）
3. 猎头给的参考价（猎头一般会给区间）

经验法则（Go 高级工程师，一线互联网）：
P6: 40-60万年包
P7: 60-100万年包
P8: 100-180万年包
```

### 3.2 第二步：设置锚点

**锚点效应**：先开口的人设定基准线。

**错误做法**：
> HR："你期望多少？" → 你说"30K" → HR 按 30K 谈

**正确做法**：
> HR："你期望多少？" → 你说"基于我的经验，市场同等水平应该在 45-50K 左右" → HR 按这个区间谈

**话术**：
> "基于我目前的薪资结构和新工作的挑战，我希望争取到 年包 80 万左右。如果有竞争力我会认真考虑。"

不要说死，给自己留余地。

### 3.3 第三步：用竞争 Offer

这是最强筹码，但有技巧：

```
✅ 正确：
"我目前也在看其他几家，有一家给了 XX 万年包，但我更倾向咱们这边的业务方向。如果能到 YY，我可以尽快确认。"

❌ 错误：
"你不给我到 XX 我就去别家了"（威胁感太强）
```

### 3.4 第四步：谈 Base 优先

Base 是最影响长期收入的部分（税高但稳定）。

```
如果 Base 谈不上去：
→ 争取更多 Sign-on（一次性，税低）
→ 争取更高 Bonus 比例
→ 争取更多 RSU
```

## 4. 常见问题应对

### Q：HR 说"我们有自己的薪资体系，没办法再加了"

> "我理解，不过我了解到同级别的市场水平是 XX，如果这边有困难，能否在 Sign-on 或者 RSU 上做一些补充？"

### Q：HR 压价说"你经验不够 P7 级别"

> "我理解公司对级别的要求，我来举几个具体的例子说明我达到 P7 的依据：1）... 2）... 3）... 如果公司认为我在某些方面还有欠缺，我愿意接受试用期的考核。"

### Q：如何谈试用期薪资？

> 试用期薪资不得低于正式薪资的 80%，可以谈。

### Q：什么时候该 Accept？

> 当：
> 1. 总包达到你预期（或接近）
> 2. 发展机会合适
> 3. 没有其他重要 Offer 在等
>
> 不要为了多 5% 拖太久（1-2 周合理），拖太久可能 Offer 被撤。

## 5. 跳槽薪资涨幅参考

| 情况 | 合理涨幅 |
|------|---------|
| 平级跳（大公司→大公司）| 10-20% |
| 升职跳（P6→P7）| 30-50% |
| 创业公司（风险高）| 靠期权翻盘 |
| 降级跳（大公司→小公司）| 靠现金或期权补偿 |

> 注意：涨幅不是绝对的，有时候平薪跳也值得（比如更好的技术方向、更稳定的环境）。

## 7. 注意事项

1. **不要只盯着月薪**：总包才是真实收入
2. **福利也要算**：免费三餐、房补、补充公积金都是钱
3. **签字前确认**：签字后很难反悔
4. **背景调查**：谈的时候可以适当美化但不要造假，背景调查会被查
5. **Offer 有效期**：注意 Deadline，拖太久 Offer 可能失效
